- dexie
- yarn
- IndexedDB

## How to run
1. yarn add
2. yarn start
3. go to http://localhost:6075
4. other pages
  - online IDE: http://localhost:6076
  - dev server: http://localhost:6077

### For deployment
1. yarn build
2. yarn deploy (run in server)
3. use `serve` to serve static files
